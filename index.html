Google Sheets Auto-Sync
Sync all POS data to your Google Sheet automatically
1Create your Google Sheet

Go to sheets.new to create a new Google Sheet. Copy the Sheet ID from the URL.

https://docs.google.com/spreadsheets/d/COPY-THIS-PART/edit

2Set up Google Apps Script (one time only)
  1. Open your Google Sheet → Extensions → Apps Script
  2. Delete everything, paste the script below, click Save
  3. Click Deploy → New deployment → type: Web app
  4. Set "Execute as" = Me, "Who has access" = Anyone
  5. Click Deploy, authorize, then copy the Web App URL
3Sync your data
